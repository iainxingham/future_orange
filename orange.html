Notes
  • This model will break down if so few people remain doing nights that it is not possible to keep the number of weekends even across both groups. This will show as a negative number of daytime weekends on the nights rota column.
  • A weekend day is assumed to be 4 PA with any additional time "on call"
  • A 42 week working year is assumed with all out of hours work annualised
  • All emergency return hours are assumed to be in premium time
